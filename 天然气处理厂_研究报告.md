## 摘要

天然气处理厂是天然气产业链中承上启下的关键环节，承担着将气井产出的原料天然气净化、脱水、提取轻烃产品、确保气质达标外输的核心任务。本报告从工艺制程、生产组织、质量管理三大维度，系统梳理天然气处理厂的运营管理体系，为行业认知和数字化建设提供知识基础。

## 第一章 工艺制程

### 1.1 总体工艺流程

天然气处理厂的典型全流程包含六个核心环节：

```
原料气进站 → 入口分离 → 脱硫脱碳 → 脱水 → 凝液回收(NGL) → 产品气增压外输
                                        ↓
                              硫磺回收 + 尾气处理
```

**全流程特点**：
- **连续化生产**：全年365天不间断运行，设计运行周期通常为3~5年（大修间隔）
- **高压操作**：系统操作压力从3.0MPa到12.0MPa不等
- **深冷操作**：NGL深冷回收温度可达-70℃至-100℃
- **有毒介质**：涉及H₂S（剧毒）、NH₃、甲醇等危险化学品

### 1.2 入口分离（气液分离）

**目的**：脱除原料气中的游离水、凝析油、固体杂质。

**核心设备**：
- 段塞流捕集器（Slug Catcher）：处理气液两相段塞流
- 三相分离器：同时分离气、油、水三相
- 过滤分离器：精细过滤固体颗粒

**操作参数**：
| 参数 | 典型范围 |
|------|---------|
| 操作压力 | 3.0 ~ 12.0 MPa |
| 操作温度 | 10 ~ 60℃ |
| 分离效率 | >99%（液滴去除） |

### 1.3 脱硫脱碳（酸性气体脱除）

脱硫是天然气处理中最关键的工序，直接决定产品气安全性和环保合规性。

**主流工艺对比**：

| 工艺 | 吸收剂/介质 | 适用场景 | 优势 | 局限 |
|------|-----------|---------|------|------|
| 醇胺法（MDEA） | N-甲基二乙醇胺 | 高碳硫比气体 | 选择性好、能耗低 | 胺液降解、发泡 |
| Sulfinol法 | 环丁砜+胺液 | 高酸气浓度 | 处理深度高 | 溶剂成本高 |
| Selexol法 | 聚乙二醇二甲醚 | 高压高CO₂ | 物理吸收、低能耗 | 低温操作 |
| Rectisol法 | 冷甲醇 | 深度净化 | 极高净化度 | 设备复杂、-60℃ |
| 膜分离法 | 聚合物膜 | CO₂预处理 | 无化学品、模块化 | 精度有限 |

**醇胺法典型流程**：
```
原料气 → 吸收塔（逆流接触MDEA胺液）→ 净化气（H₂S≤6mg/m³）
富液 → 闪蒸罐（释放溶解烃）→ 贫富液换热器 → 再生塔（120℃蒸汽汽提）
再生塔顶 → 酸气（H₂S+CO₂）→ 硫磺回收
贫液 → 循环泵 → 吸收塔
```

**中国典型高含硫处理厂**：
- 普光净化厂：处理H₂S含量高达15%的原料气，采用MDEA+尾气处理，硫回收率>99.8%
- 元坝净化厂：H₂S含量约5%，采用类似工艺路线

### 1.4 脱水（水分脱除）

**方法对比**：

| 方法 | 原理 | 水露点降 | 适用场景 |
|------|------|---------|---------|
| TEG甘醇脱水 | 三甘醇吸湿 | 40~70℃ | 常规管输气 |
| 分子筛脱水 | 3A/4A分子筛吸附 | >100℃ | 深冷NGL/LNG |
| 氯化钙脱水 | 化学吸湿 | 30~50℃ | 边远小型站 |

**TEG脱水**：占天然气脱水市场的60%以上，运行成本低，操作简便。  
**分子筛脱水**：用于深冷工艺前的深度脱水，水露点可降至-100℃以下。

### 1.5 凝液回收（NGL Recovery）

**工艺分类**：

| 工艺 | 制冷方式 | 操作温度 | 回收率 | 投资水平 |
|------|---------|---------|--------|---------|
| 浅冷分离 | 丙烷制冷 | -25~-35℃ | C₃ 70~85% | 中等 |
| 深冷分离（膨胀机） | 等熵膨胀 | -70~-100℃ | C₂ 85~95%, C₃ 99%+ | 较高 |
| 混合制冷剂 | 甲烷+丙烷混合 | -80~-110℃ | C₂ 90%+ | 高 |

**深冷NGL回收核心设备**：
- 膨胀机（Expander）：等熵膨胀制冷，同时回收功
- 脱甲烷塔：分离甲烷与C₂⁺组分
- 冷箱（Cold Box）：板翅式换热器，多股流换热

**NGL分馏产品**：
```
混合NGL → 脱乙烷塔 → 乙烷产品
         → 脱丙烷塔 → 丙烷/LPG
         → 脱丁烷塔 → 丁烷/LPG
         → 天然汽油（C₅⁺）
```

### 1.6 硫磺回收与尾气处理

**Claus硫磺回收**：
- 热反应段：1/3 H₂S燃烧为SO₂
- 催化段（2~3级）：2H₂S + SO₂ → 3S + 2H₂O
- 单程回收率：94~97%

**尾气处理方案对比**：

| 工艺 | 技术路线 | 总硫回收率 | 投资/运行成本 |
|------|---------|-----------|-------------|
| SCOT | 尾气加氢+吸收 | >99.8% | 高/中 |
| SuperClaus | 选择性催化氧化 | >99.0% | 中/低 |
| CANSOLV | 胺液吸收 | >99.9% | 高/中 |

### 1.7 辅助系统

| 系统 | 功能 | 关键设备 |
|------|------|---------|
| 火炬系统 | 紧急放空与安全泄压 | 高架火炬、地面火炬 |
| 燃料气系统 | 为加热炉/锅炉/燃气轮机供气 | 调压阀、过滤器 |
| 热媒系统 | 提供工艺热源 | 导热油炉、蒸汽锅炉 |
| 冷却系统 | 工艺介质冷却 | 空冷器、水冷器 |
| 仪表风/氮气 | 控制阀驱动、吹扫惰化 | 空压机、制氮机 |
| 污水处理 | 生产废水处理回注 | 隔油池、气浮、生化 |

## 第二章 生产组织

### 2.1 组织架构

天然气处理厂采用"厂级—车间级—班组级"三级管理模式：

```
厂级管理层
  ├─ 生产技术部（工艺/设备/仪表）
  ├─ 安全环保部（HSE）
  ├─ 质量管理部（化验/质检）
  ├─ 综合管理部（人事/财务/行政）
  └─ 调度指挥中心
        ├─ 集气站/首站
        ├─ 净化/处理装置车间
        ├─ NGL回收/分馏车间
        ├─ 硫磺回收车间
        ├─ 公用工程车间
        └─ 外输计量站
```

### 2.2 人员配置

| 处理厂规模 | 日处理能力 | 常驻人员 | 倒班模式 |
|-----------|-----------|---------|---------|
| 小型厂 | <100万Nm³/d | 50~100人 | 三班两倒 |
| 中型厂 | 100~500万Nm³/d | 100~200人 | 四班三倒 |
| 大型厂 | 500~2000万Nm³/d | 200~500人 | 四班两倒 |
| 超大型厂 | >2000万Nm³/d | 500~1000人 | 四班两倒+专业班组 |

### 2.4 倒班制度

**四班三倒制**（行业最常用）：
- 每班8小时：白班(08:00~16:00)、中班(16:00~00:00)、夜班(00:00~08:00)
- 3个班组在岗，1个班组轮休
- 8天一个完整循环

**交接班六交清**：
1. 产量交清 — 当日处理量、产品产量
2. 质量交清 — 产品气质指标、异常
3. 设备交清 — 运行/备用设备状态
4. 安全交清 — 安全隐患、环保排放
5. 工具交清 — 工器具、钥匙
6. 指令交清 — 上级指令、注意事项

| 深冷设备 | 冻伤、脆断 | 低温材料、保温 |

**应急预案三级体系**：
| 级别 | 预案类型 | 演练频次 |
|------|---------|---------|

### 3.2 产品质量标准

**管输天然气（GB 17820-2018）**：

| 项目 | 一类气 | 二类气 | 三类气 |
|------|--------|--------|--------|
| 高位发热量（MJ/m³） | ≥36.0 | ≥31.4 | ≥31.4 |
| 总硫（mg/m³） | ≤20 | ≤100 | ≤200 |
| H₂S（mg/m³） | ≤6 | ≤20 | ≤350 |
| CO₂（%） | ≤2.0 | ≤3.0 | — |
| 水露点 | 低于最低管输温度 | | |

**LNG原料气典型要求**：

| 项目 | 典型限值 |
|------|---------|
| H₂S | ≤4 ppmv |
| CO₂ | ≤50 ppmv |
| H₂O | ≤0.1 ppmv（-100℃露点） |
| 汞 | ≤0.01 μg/Nm³ |

**LPG产品（GB 11174-2011）**：

| 项目 | 一级品 | 二级品 |
|------|--------|--------|
| 烯烃含量（%） | ≤2.0 | ≤3.0 |
| C₅+含量（%） | ≤1.0 | ≤2.0 |
| H₂S（mg/m³） | ≤5 | ≤10 |

**硫磺产品（GB 2449-2022）**：

| 项目 | 优等品 | 一等品 |
|------|--------|--------|
| 硫含量（%） | ≥99.90 | ≥99.50 |
| 水分（%） | ≤0.10 | ≤0.50 |
| 灰分（%） | ≤0.03 | ≤0.10 |

```
第一阶段：自动化（DCS+ESD+在线分析仪）

第二阶段：信息化（LIMS+MES+EAM+ERP集成）
  ↓
第三阶段：数字化（数字孪生+大数据+AI辅助决策）
  ↓
第四阶段：智能化（自主优化+预测维护+智能调度）
```

3. **模块化建设**：橇装化、工厂预制，缩短建设周期
4. **数字化交付**：智能P&ID、数字孪生、实时数据接入
5. **绿色低碳**：电气化替代、余热利用、碳减排
